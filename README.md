# Learning Go

A collection of Go code for learning the language.

## Projects

### bank/
A simple banking application that demonstrates:
- User input handling with `fmt.Scan`
- File I/O operations (reading/writing balance to file)
- Control flow with switch statements and loops
- Custom packages (`fileops`)
- Using external packages (`go-randomdata`)

### investment/
An investment calculator that computes future value of investments, covering:
- Mathematical operations with the `math` package
- Functions with multiple return values
- Constants and variables
- Formatted output with `fmt.Sprintf`

### profit-calculator/
A profit calculator that computes earnings before tax and profit margins, demonstrating:
- Functions with named return values
- Basic arithmetic operations
- User input handling

### pointers/
A demonstration of pointers in Go, covering:
- Pointer declaration and initialization
- Address-of operator (`&`)
- Dereferencing pointers (`*`)
- Passing pointers to functions
